Section 2 - Types

Specificity

Specificity applies to the different implementations of CSS, and states that if a property of an element is defined more than once, the browser will use the most specific version.

This means that properties defined in inline CSS take priority over internal CSS and as such, it is advisable to only use one implementation of CSS. Most people choose to use external CSS as it is the easiest to use, with one stylesheet responsible for a whole website.
