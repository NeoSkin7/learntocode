Section 5 - Units

Relative Units

Relative units are units that change relative to another length property. This makes them more difficult to use but better for responsive design because they are scalable. The nine absolute units are as follows:

  • em - Relative to the font size of the element (2em means 2 times the current font size)
  • ex - Relative to the x-height of the current font (rarely used)
  • ch - Relative to the width of the character "0" (ch stands for character)
  • rem - Relative to the font size of the root element (html)
  • vw - 1% of the width of the browers viewport (window size) (vw stands for viewport width)
  • vh - 1% of the height of the browers viewport (vh stands for viewport height)
  • vmin - 1% of the smaller viewport dimension (vmin stands for viewport minimum)
  • vmax - 1% of the larger viewport dimension (vmin stands for viewport maximum)
  • % - Relative to the parent element (2% is 2% of the parent element)
