Section 5 - Units

Units!!

Units are a basic concept in CSS and help you to define lengths and sizes of elements. You've already run into a few in Section 3 and probably wondered what they meant.

Absolute Units

Absolute units are direct translations of real-life lengths into CSS. This makes them easy to use as they are already a well-defined concept. The six absolute units are as follows:

  • cm - Centimeters
  • mm - Millimeters
  • in - Inches (1in = 96px = 2.54cm)
  • px - Pixels (1px = 1/96th of 1in)
  • pt - Points (1pt = 1/72 of 1in)
  • pc - Picas (1pc = 12 pt)

The only absolute unit that is usually used is pixels, but for responsive design even that is usually dropped.
