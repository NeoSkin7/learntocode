CSS ESSENTIALS

Section 8 - Advanced CSS

There are many things that have been omitted here; the box model, flexbox, more selector types, and much more.

Here are a few more properties you can look into before you reach the Advanced CSS course:

list styling; the list-styling-type changes the type of list markers used, list-styling-image replaces the markers with images.

text-align, aligns the text to a specific horizontal position.

border, creates a border around an element that can be heavily customised.

float, positions elements absolutely on a specified horizontal position without needing parent element position styles.

A list of more CSS properties and values is available on the Mozilla Developer Network page.
