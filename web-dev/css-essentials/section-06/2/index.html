Section 6 - Colors

HEX Colors

HEX Values (or codes) are the final (commonly-used) way to define colors in CSS. They work similarly to RGB as in the example below.

Code Snippet

#00FFFF00

This HEX color code is for red; the first number is red, second - green, and third - blue. The values range from 00 (no color) to FF (full color). The last number defines opacity and has the same range of values.

The HEX range of values is below (as it is a little complex). Each 2-digit string represents a possible combination for each color in HEX (red, green, blue, or opacity). 

00 01 02 03 04 05 06 07 08 09 0A 0B 0C 0D 0E 0F ... 50 ... 5A ... A0 ... AA ...

Some values omitted.

You can also write HEX codes as a string of 3, 4, or 6 values (#RGB, #RGBO, #RRGGBB).
