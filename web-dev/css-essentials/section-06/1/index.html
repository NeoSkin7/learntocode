Section 6 - Colors

RGB Colors

RGB Values are another way to define colors in CSS. Simply dial in the amount of red, green, and blue and you're good to go!

Code Snippet

rgb(255, 255, 255)

This RGB color code is for white; the first number is red, second - green, and last blue. The values range from 0 (no color) to 255 (full color).

There is also RGBA which allows you to define alpha (opacity) like in the code below. 

rgba(255, 255, 255, 1.0)

This RGBA color code is for white; the last number defines alpha (or opacity) and ranges from 1 (opaque) to 0 (transparent).
