CSS ESSENTIALS

Preface

During this course, you will need to have a place to code; to create a file to code in, make a new text document and name it as (document name).css, making sure to highlight all existing text when renaming the file. Next, connect it to a HTML document created using the instructions from the last course (Using Preface with base content from Section 9), you can do this by inserting this code into the head section of the HTML document:<link rel="stylesheet" href="(document name).css">.

When asked to enter code throughout the course, right click on the file you've been instructed to edit (either the HTML or CSS one) and select Open With > Notepad (or Open With > Choose another app > Notepad if Notepad is unavailable in the normal list).

If you need more help, try typing out the code snippets into the CSS file created above to see how they display.
