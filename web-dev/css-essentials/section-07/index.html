CSS ESSENTIALS

Section 7 - Pseudo Classes & Elements

Pseudo Classes

Pseudo Classes select specific states of an element. For example, an anchor element has hover and link pseudo classes.

Pseudo classes are "tacked" onto normal selectors using a colon, for example an anchor tag with a hover pseudo class is written like this: a:hover.

Pseudo Elements

Pseudo elements are based on the same principle as pseudo classes and are designed to select specific parts of an element, like before it or after it.

Pseudo elements are "attached" to normal selectors using a double colon, for example an anchor tag with a before pseudo element is written like this: a::before.
