Introduction - How The World Wide Web Works

The world wide web works on a "net" of servers that are all interconnected in order to serve content to the end user, this system allows for any one of millions of different locations to request data from a single server.

So how exactly can a user interface with this system? To put it simply, the user requests a webpage by entering a URL into their web-browsers omnibox, the web-browser then sends that request to their router which passes it to the users ISP.

The ISP then searches for that URL in their DNS (or the one manually set by the user) which "outputs" an IP address for the given server. If found, the ISP contacts the server using that IP address for the given URL with the IP address of the users router and requested page.

Finally, the server sends back the response to the users router which then forwards it to the correct computer, where it is displayed in their web-browser.
