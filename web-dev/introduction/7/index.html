Introduction - Evaluating Languages

You may already feel overwhelmed at the prospect of so many different languages that you can learn. However, they all serve different functions and, when combined, can create a seamless
web experience.

So how to know which language is best for your particular task?
It's simple really, you just need to evaluate its skillset and decide whether it matches your use-case. For example, JavaScript is designed for user viewed operations, such as a popup. Remember that if something is commonly used for a task then it's a good idea to "adopt" it in your work.
