HTML ESSENTIALS

Section 1 - Elements & Tags

Elements

An element is the most basic concept in HTML, they comprise of a word or phrase (sometimes shortened to a letter) with a specific meaning. For example, h1 is the largest heading element (mentioned later).

There are no fingerspaces in an element.

Tags

Elements make up tags which surround content that will be shown on your page. Elements enclosed with a left angle bracket (<) and a right angle bracket (>) are tags.

An opening tag (<element>) is always followed by a closing tag (</element>).

Code Snippet

<p>Hello World!</p>

This shows an opening and closing tag in use.
