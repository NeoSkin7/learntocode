Section 4 - Other Elements

Self-closing Tags

img is the image tag, it displays images.

hr is the horizontal rule tag, it creates a horizontal line across the page.

br is the line break tag, it starts a new line of text.

The above tags do not require a closing tag, this is because they do not contain any content.

Code Snippet

<img src="https://example.com/image.svg">

This code would display the specified image.

Self-closing tags are sometimes written with a forward-slash before the closing right-angle bracket. For example, the above example would be written as below. Both ways are valid. 

<img src="https://example.com/image.svg" />
