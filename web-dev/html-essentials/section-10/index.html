HTML ESSENTIALS

Section 10 - Advanced HTML

There are many more HTML elements that have been omitted here, below is a list of just a few that you should know:

div, divides separate content sections. You can also use these elements to achieve the same effect for different content: header, nav, footer, section, and article.

span, allows for the styling of text for highlighting and changes in color.

button, creates clickable content, a can also be styled for the same purpose.

A full list of of HTML elements is available on the Mozilla Developer Network page.
