HTML ESSENTIALS

Section 8 - Forms

Code Snippet

<form action="mailto:jdoe@example.com" method="POST">
  <label>Text Input:</label>
  <input type="text" value="text" required>
  <label>Checkbox:</label>
  <input type="checkbox">
  <input type="submit" value="Submit">
  <input type="reset" value="Reset">
</form>

This code would display a form with a text input field, a checkbox, and reset and submit buttons.
