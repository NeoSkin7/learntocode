Section 8 - Forms

How to Create a Form

  1. Define a form with the form tag.
  2. Specify in the form element where to send form data with the action attribute.
  3. Specify in the form element how to send form data with the method attribute.
  4. Provide a way for users to input and submit responses with the input element.
  5. Specify in the input element what input type you need using the type attribute.
  6. Label input fields with the label element (repeat steps 4-6 until you have all desired inputs for your form).
  7. Create anther input tag and set the type attribute equal to submit.
