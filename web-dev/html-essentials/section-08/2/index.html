HTML ESSENTIALS

Section 8 - Forms

Form Input Attributes

type is the type of input attribute, it allows you to specify the correct input type to use.

value is the initial value of the input field attribute, it allows you to specify the initial text an input field holds.

placeholder is the placeholder text of the input field attribute, it allows you to specify the placeholder (example text) of an input field.

required is the required input attribute, if present, it makes a input field required.

Form Attributes

action is the action form attribute, it specifies where to send form data once submitted.

method is the method form attribute, it specifies how to send form data once submitted.
