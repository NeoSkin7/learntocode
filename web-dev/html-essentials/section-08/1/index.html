Section 8 - Forms

Form Input Types

text is the text input attribute, it allows the user to input a small amout of text.

checkbox is the checkbox input attribute, it displays a box the user is able to check (tick).

date is the date picker input attribute, it allows the user to input a date.

file is the file upload input attribute, it allows the user to upload a file from their computer.

range is the range input attribute, it allows the user to use a slider.

number is the number input attribute, it allows the user to input a numerical value.

reset is the reset button input attribute, it allows the user to reset all input fields within the form.

submit is the submit button input attribute, it allows the user to submit the form.
