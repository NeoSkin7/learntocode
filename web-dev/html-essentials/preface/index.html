HTML ESSENTIALS

Preface

During this course, you will need to have a place to code; to create a file to code in, make a new text document and name it as (document name).html, making sure to highlight all existing text when renaming the file. When you open it, it will display in your web browser.

When asked to enter code throughout the course, right click on the file and select Open With > Notepad (or Open With > Choose another app > Notepad if Notepad is unavailable in the normal list).

If you need more help, try typing out the code snippets into the file created above to see how they display, you can also try combining some elements if you're feeling adventurous!
