HTML ESSENTIALS

Section 5 - Lists

Order or Chaos?

ul is the unordered list tag, it creates bulleted lists.

ol is the ordered list tag, it creates numbered lists.

li is the list item tag, it siginifies a new number or bullet point.

You must define every bullet point or number with an li tag.

Code Snippet

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>

This code would display a bulleted list with two items.
