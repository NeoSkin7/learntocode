HTML ESSENTIALS

Introduction

HTML is the mother of all modern website coding languages and was invented in 1990 by Tim Berners-Lee. Since then it has undergone 5 different updates to introduce new features and compatibility to the language.

The language is compatible with all web-browsers natively as it was created specifically for this use.

In 1995, the coding languages JavaScript and PHP were created to add scripting to HTML, they are most commonly used with the language SQL.

Then in 1996, the language CSS was invented to enable further styling of websites.

This course covers the basics of HTML; teaching you about elements, tags, and attributes that you can use to create a simple website.
